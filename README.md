## Deployment

Please run the following commands to host a server:

```
git clone https://github.com/jkjkjiin/test.git
cd Space
npm i
npm run build
node server.js
```
### setting up proxy server

```
npm i
npm run build
node server.js
```
### To setup proxy dev server
```
npm i
npm run dev
```
### Start proxy server

```
node server.js
```
### Updating

```
git pull --force --allow-unrelated-histories
```
